## Components 组件库
> 自定义组件

### 说明
- 系统自定义的组件库，前提需满足以下条件之一，相反没必要进入组件库
  - 经常使用
  - 逻辑相对复杂

### 要求
- 组件内部数据，必须以 `prop` 形式传入，不在组件内部直接使用系统特定的数据源，例如: 
  - 特定接口
  - 全局属性
  - 共享状态
  - ...

- **封装**：提取通用逻辑和功能，一旦确认为共用组件，禁止直接修改组件代码，实现封装、一致
- **可扩展**: 鉴于可能大部分组件依赖于第三方组件库，不能屏蔽第三方组件主要的 props、methods 等，可以再次整合调整
- 卡槽 (slot): 可选
